#  Todo App

A modern Flutter Todo application built with a clean architecture and reusable widgets.  
Users can create tasks, organize them with colors, track their status, and manage daily productivity through a simple and elegant interface.

---

#  Features

-  Profile Setup Screen
-  Home Dashboard
-  Add New Task
-  Task Description
-  Task Color Selection
-  Task Status (Pending / Done)
-  Task Statistics
-  Clean & Modern UI
-  Reusable Widgets
-  Responsive Layout
